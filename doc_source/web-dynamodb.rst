######################
Working with |DDBlong|
######################

.. meta::
    :description: Hosting .NET applications on AWS using Amazon DynamoDB
    :keywords: .net, guide, help, tutorial, dynamodb, web, scenarios

|DDBlong| is a fast and flexible NoSQL database service for all applications that need consistent, single-digit
millisecond latency at any scale. It's a fully managed cloud database and supports both document and key-value
store models. Its flexible data model, reliable performance, and automatic scaling of throughput capacity make it
a great fit for web and other applications.
